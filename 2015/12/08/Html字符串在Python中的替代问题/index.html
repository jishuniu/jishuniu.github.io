Html字符串在Python中的替代问题

2015.12.08 Landarky 笔记

问题描述

  • 在Python中，有时需要替换HTML格式的字符串，由于%(name)s是替换字符串的，所以%被占用，有时会遇到以下问题

解决方案

  • 这是因为HTML代码中出现了%, 如90%这些参数在HTML中很常见，需要改成90%%, 这样就可以识别而不报错了
